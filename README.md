# 💬 CHATSITO - Simple Group Chat

Una aplicación de mensajería ligera que permite la comunicación organizada por grupos con persistencia de datos local.

## 🔗 Demo en vivo
Prueba el servicio aquí: [https://porto2710-lab.github.io/chat/](https://porto2710-lab.github.io/chat/)

## 🚀 Características
* **Múltiples Grupos:** 6 canales de conversación independientes.
* **Persistencia Local:** Tus mensajes se guardan en el navegador (LocalStorage).
* **Interfaz Moderna:** Diseño inspirado en Discord con soporte para modo oscuro.
* **Scroll Automático:** La ventana de chat siempre muestra el mensaje más reciente.

## 🛠️ Stack Tecnológico
* **HTML5:** Estructura semántica para la interfaz de chat.
* **CSS3:** Estilos avanzados, Flexbox y temas oscuros.
* **JavaScript (ES6):** Lógica de almacenamiento, filtrado por grupos y manipulación del DOM.
